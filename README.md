# Scenario-to-Arabic-Poetry
# Arabic Poetry Dataset

This repository contains an Arabic poetry dataset prepared for NLP and AI projects.

## File
- `arabic_poetry_dataset.json`

## Dataset Description
The dataset contains Arabic poetic verses linked with meanings or scenarios.

## Example

```json
{
  "scenario": "الحزن",
  "poem": "وللصبر في بعض المواطن عزة"
}
```

## Usage
This dataset can be used for:
- NLP projects
- Fine-tuning language models
- Retrieval-Augmented Generation (RAG)
- Arabic poetry analysis
